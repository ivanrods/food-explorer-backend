
# Food Explorer

O food explorer, um menu interativo para um restaurante fictício.


## Autores

- [@ivanrods](https://github.com/ivanrods)


## Funcionalidades

- Login
- Cadastro
- Detalhes do prato
- Novo prato
- Editar prato
- Favoritos


## Instalação

Instale my-project com npm

```bash
  npm install
  npm run migrate
  npm start
```
